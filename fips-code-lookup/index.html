What is a FIPS code

FIPS (Federal Information Processing Standard) county codes are 5-digit numbers assigned to every county and county-equivalent in the US. The first 2 digits are the state FIPS code; the last 3 are the county code within that state. For example, Los Angeles County, California is 06037 (06 = California, 037 = Los Angeles County).

FIPS codes are used in US Census data, federal grant applications, CDC health data, USDA programs, HUD housing data, environmental regulatory filings, and many other federal systems.

State FIPS codes and combined identifiers

Some federal systems use the 2-digit state FIPS code separately from the 3-digit county FIPS code; others require the combined 5-digit code. The app shows all three values: the 2-digit state code, the 3-digit county code, and the combined 5-digit FIPS code.

For census tract and block-level analysis, the county FIPS code is the prefix to the tract code. The app's county-level lookup is the starting point for these more granular geographic identifiers.
